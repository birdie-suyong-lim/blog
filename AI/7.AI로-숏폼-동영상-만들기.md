## 들어가며 — "촬영하지 않고 영상을 만든다"

2024년만 해도 "AI로 영상을 만든다"는 말은 데모 수준이었다. Sora 발표 영상이 화제가 됐지만 일반인은 못 썼고, Runway·Pika 같은 도구도 짧고 어색한 클립만 뽑아냈다.

2026년 현재는 다르다.
- **OpenAI Sora 2**가 일반 사용자에게 풀렸고 (ChatGPT 구독에 포함)
- **Google Veo 3**는 오디오까지 동시 생성하고
- **Kling·Hailuo** 같은 무료급 도구가 사실성에서 놀라운 수준에 도달했고
- **Opus Clip**은 긴 영상을 알아서 숏폼으로 잘라준다

그러나 도구만 늘었을 뿐 **"숏폼이 잘 되는 공식"** 자체는 여전하다. 짧고, 첫 1초가 강하고, 자막이 잘 보이고, 후크가 분명해야 한다. AI는 이 공식을 자동화하는 데 강하지, 공식 자체를 대신해주지 않는다.

이 글은 ① AI 영상 도구의 5가지 분류, ② 도구별 강·약, ③ 콘텐츠 유형별 워크플로우, ④ 플랫폼 규격과 함정을 정리한다.

## 1. AI 영상 도구, 5가지 분류

| 분류 | 대표 도구 | 입력 | 결과물 |
|------|----------|------|--------|
| **A. Text-to-Video** | Sora 2, Veo 3, Runway, Luma, Pika, Kling | 텍스트 프롬프트 | 5~10초 클립 (AI가 통째로 생성) |
| **B. AI 아바타·토킹헤드** | HeyGen, Synthesia, D-ID | 스크립트 + 아바타 선택 | 사람이 말하는 영상 |
| **C. 롱폼 → 숏폼 클리핑** | Opus Clip, Vizard, Vidyo.ai | 긴 영상 (유튜브 URL 등) | 1~3분짜리 숏폼 후보 N개 |
| **D. AI 편집 어시스턴트** | CapCut, Descript, Premiere + Firefly | 직접 촬영한 영상 | 자동 자막·하이라이트·B-roll |
| **E. 모션 슬라이드·템플릿** | Canva Video, CapCut 템플릿 | 텍스트·이미지 | 정보형 모션그래픽 |

→ **대부분의 SNS 운영자는 C·D 위주로 시작**한다. A는 미학·실험용, B는 비즈니스 설명용에 가깝다.

## 2. 분류별 주요 도구

### A. Text-to-Video 모델들

#### OpenAI Sora 2
- 2025년 말부터 일반 사용자 접근 가능 (ChatGPT 구독에 포함)
- **물리 시뮬레이션**과 캐릭터 일관성이 다른 도구보다 강함
- 1080p, 최대 20초 클립
- 한국어 프롬프트도 잘 이해
- [openai.com/sora](https://openai.com/sora)

#### Google Veo 3
- **오디오를 함께 생성** (대사·환경음·음악)
- 영화적 카메라 무브 우수
- Google AI Premium 또는 Workspace에서 사용
- [deepmind.google/veo](https://deepmind.google/technologies/veo/)

#### Runway Gen-4
- 영화 제작·뮤직비디오 업계가 가장 많이 쓰는 도구
- **캐릭터·로케이션 레퍼런스**를 등록해 컷 간 일관성 유지
- $15~95/월
- [runwayml.com](https://runwayml.com)

#### Luma Dream Machine (Ray2)
- 가성비 좋고 빠름. 5초 클립을 1~2분 안에 뽑음
- $9~$32/월, 무료 플랜 있음
- [lumalabs.ai](https://lumalabs.ai)

#### Pika 2.0
- 인페인팅·확장·카메라 무브 컨트롤이 직관적
- 짧고 재밌는 클립에 강함
- $10~$70/월
- [pika.art](https://pika.art)

#### Kling AI / Hailuo (중국)
- **무료 플랜 한도가 매우 큼** — 입문자가 가장 만져보기 좋음
- 사실성·움직임 자연스러움이 놀랍게 좋음
- 한국 사용자도 많음
- [klingai.com](https://klingai.com), [hailuoai.video](https://hailuoai.video)

### B. AI 아바타·토킹헤드

#### HeyGen
- **스크립트만 입력하면** 미리 선택한 아바타가 영상으로 말함
- 한국어 음성·립싱크 자연스러움 최상위
- 본인 얼굴·목소리 클로닝 가능 (자기 아바타 만들기)
- $24~$89/월
- [heygen.com](https://www.heygen.com)

#### Synthesia
- 기업 교육·내부 커뮤니케이션에 표준급
- 140개 이상 언어 지원
- $29~$89/월
- [synthesia.io](https://www.synthesia.io)

#### D-ID
- 사진 한 장 → 말하는 영상 (기본형)
- 빠르고 저렴
- [d-id.com](https://www.d-id.com)

### C. 롱폼 → 숏폼 클리핑

#### Opus Clip — 가장 대중적
- 유튜브 URL 또는 영상 업로드 → AI가 **하이라이트 후보 10~30개** 자동 추출
- 각 클립에 자동 자막·줌인·세로 변환(9:16) 자동 적용
- **바이럴 점수**까지 매겨줌 (어떤 클립이 클릭률 높을지 예측)
- $9~$29/월
- [opus.pro](https://www.opus.pro)

#### Vizard
- Opus Clip의 강력한 경쟁자. 한국어 자막 정확도 우수
- 무료 플랜 한도 적당
- [vizard.ai](https://vizard.ai)

#### Vidyo.ai
- 비슷한 카테고리, 가성비 위주
- [vidyo.ai](https://vidyo.ai)

### D. AI 편집 어시스턴트

#### CapCut
- **전 세계 숏폼 크리에이터 표준 도구**. 모바일·데스크탑 모두
- 자동 자막(한국어 매우 정확), 자동 하이라이트, AI 보이스오버, 트렌드 음악 라이브러리
- 무료로도 충분히 강력. Pro $7.99/월
- 모회사가 틱톡과 같은 ByteDance — 틱톡 트렌드와 직결
- [capcut.com](https://www.capcut.com)

#### Descript
- **텍스트 기반 영상 편집** — 스크립트를 수정하면 그 부분 영상이 잘림
- 팟캐스트·인터뷰·강의 영상에 압도적
- "Overdub"으로 본인 목소리 클로닝 → 잘못 말한 부분을 텍스트 수정으로 자연스럽게 교체
- $19~$50/월
- [descript.com](https://www.descript.com)

#### Adobe Premiere Pro + Firefly Video
- 프로 NLE에 AI 기능 통합 (텍스트 기반 편집, 자동 자막, 생성형 채우기)
- 기존 영상 편집자에게 가장 자연스러운 진입
- Creative Cloud 구독

#### DaVinci Resolve (무료)
- 무료 NLE 중 최강. 색보정·VFX는 업계 표준급
- AI 기능 (얼굴 보정·자동 트래킹·음성 분리)이 내장
- [blackmagicdesign.com/products/davinciresolve](https://www.blackmagicdesign.com/products/davinciresolve)

### E. 모션 슬라이드·템플릿

#### Canva Video
- 인스타·틱톡 템플릿 + 텍스트 애니메이션 + 이미지 슬라이드 영상
- AI 이미지 생성도 통합
- 정보형 콘텐츠(요리 레시피, 통계, 학습 자료) 빠르게 만들 때 강력

#### CapCut 템플릿
- 트렌드 음악·트렌드 컷에 맞춰 사진·짧은 클립만 끼워 넣으면 완성
- 입문자가 가장 빨리 결과물 얻음

## 3. 콘텐츠 유형별 워크플로우

### 워크플로우 1 — 정보형 토킹헤드 (예: "오늘의 주식 시황")

1. **스크립트 작성** (Claude/ChatGPT)
2. **HeyGen 아바타 또는 본인 클로닝**으로 영상 생성
3. **CapCut에서 자막·B-roll·BGM 추가**
4. 9:16 비율로 1080×1920 내보내기

→ 매일 같은 형식 자동화 가능. 본인 얼굴 안 보여줘도 됨.

### 워크플로우 2 — 긴 영상 → 숏폼 N개 자동 추출 (예: 유튜브 1시간 라이브 → 쇼츠 10개)

1. **긴 영상을 유튜브에 업로드** 또는 파일 보유
2. **Opus Clip에 URL 입력**
3. AI가 30분~1시간 안에 후보 10~30개 + 바이럴 점수 제공
4. 마음에 드는 클립 5~10개 선택 → 자막·줌인 자동
5. 각 플랫폼에 업로드

→ **롱폼 크리에이터에게 가장 ROI 높은 워크플로우**.

### 워크플로우 3 — 100% AI 생성 (예: 미학적 짧은 클립, 광고, 뮤직비디오)

1. **콘셉트·스토리보드** (Claude/ChatGPT로 5컷 정리)
2. **컷별 프롬프트 작성** — 카메라 무브·라이팅·스타일 명시
3. **Sora·Veo·Runway**로 컷별 클립 생성 (한 컷에 4~8개 후보)
4. **CapCut 또는 Premiere에서 컷 편집**
5. **Suno/Udio**로 BGM, **ElevenLabs**로 내레이션 (필요 시)
6. 자막·디테일 마감 → 내보내기

→ 미학 자유도 최고. 단점: 캐릭터 일관성 어려움, 시간·비용 큼.

### 워크플로우 4 — 스마트폰 촬영 + AI 마감 (가장 흔한 실전 케이스)

1. **스마트폰으로 가로/세로 막 찍기** (구도·조명 신경 안 써도 됨)
2. **CapCut으로 가져와서**:
   - 자동 자막 ON (한국어 95%+ 정확)
   - 트렌드 음악 라이브러리에서 BGM
   - 자동 하이라이트(가장 흥미로운 구간 자동 추출)
   - 컬러 그레이딩 프리셋
3. 후크 텍스트(첫 3초)만 직접 작성
4. 내보내기

→ 일상 브이로그·여행·맛집 콘텐츠는 거의 이 한 가지로 끝남.

### 워크플로우 5 — 정보형 모션 슬라이드 (예: "AI 도구 5가지")

1. **카피·구성** (Claude/ChatGPT)
2. **Canva Video 또는 CapCut 템플릿** 선택
3. 텍스트·이미지·전환 효과 입력
4. AI 보이스오버 또는 음악만
5. 내보내기

→ 얼굴·촬영 부담 없이 정보 전달. 가장 가벼움.

## 4. 플랫폼별 규격·길이 치트시트

```
YouTube Shorts
├ 비율          9:16
├ 해상도        1080 × 1920
├ 최대 길이     60초
└ 권장 길이     15~45초

Instagram Reels
├ 비율          9:16
├ 해상도        1080 × 1920
├ 최대 길이     90초 (확장 시 3분)
└ 권장 길이     7~30초 (체류 시간 우선)

TikTok
├ 비율          9:16
├ 해상도        1080 × 1920
├ 최대 길이     60초 (확장 10분)
└ 권장 길이     15~60초

Instagram 피드 동영상 (Reels 아님)
├ 비율          1:1 또는 4:5
├ 해상도        1080 × 1080 또는 1080 × 1350
└ 최대 길이     60초

YouTube 일반 동영상
├ 비율          16:9
├ 해상도        1920 × 1080 이상 (4K 권장)
└ 길이          제한 없음 (장편)
```

### 첫 1초·3초·5초 규칙
- **0~1초**: 시청자가 스크롤할지 멈출지 결정. **후크 시각 또는 후크 문장 필수**
- **0~3초**: 무엇에 관한 영상인지 명확해야
- **0~5초**: 보상이 있어야 (정보·웃음·미학·궁금증 중 하나)

AI 도구가 영상을 만들어줘도 이 구조는 **사람이 짜야** 한다.

## 5. 음성·BGM·자막의 AI

### 음성 합성 (TTS)
- **ElevenLabs** — 가장 자연스러움, 본인 목소리 클로닝 가능. $5~$330/월. 한국어 우수
- **OpenAI TTS** — ChatGPT API. 합리적 가격, 한국어 자연스러움 양호
- **HeyGen 내장 음성** — 아바타와 묶음

### AI 음악
- **Suno** — 가사·장르 입력만으로 풀 트랙 생성. 무료 일일 한도 + 유료
- **Udio** — Suno 경쟁. 음질·자연스러움 강점
- **Mubert** — 무한 스트리밍 BGM
- → **저작권 안전 라이선스 옵션 반드시 확인**

### 자막 자동 생성
- **CapCut 자동 자막** — 한국어 정확도 95%+
- **Whisper / AssemblyAI** — API로 자체 자동화 시
- **YouTube 자동 자막** — 기본 무료, 정확도는 영상 화질·발음에 좌우

## 6. 함정과 주의점

### 캐릭터·로케이션 일관성
- AI 영상은 컷마다 얼굴·옷·배경이 미세하게 달라짐 → **시청자가 즉시 알아챔**
- 해결:
  - Runway Gen-4의 "Reference" 기능으로 캐릭터·로케이션 등록
  - Sora의 "Storyboard" 모드 활용
  - 또는 한 컷 길게 사용

### 손가락·물리 오류
- 손이 6개, 컵이 공중에 뜨는 등 여전히 발생
- 해결: 손 클로즈업 피하기, 전신·중거리 컷 선호, 어색한 컷은 Generative Fill로 부분 재생성

### 길이 한계
- 대부분 AI Text-to-Video는 한 번에 **5~20초**만 생성
- 더 긴 영상은 클립을 여러 개 만들어 편집기에서 이어 붙여야 함
- **숏폼에는 오히려 잘 맞음** (15~60초가 표준이라 한 두 컷이면 충분)

### 저작권·상업 사용
- **Sora 2 / Veo 3**: 사용자가 권리 보유 (단 실존 인물·IP 묘사 제한)
- **Runway**: 상업 사용 OK (유료 플랜)
- **Kling·Hailuo**: 약관 변동성 있음 → **상업용은 반드시 약관 확인**
- **AI 음악(Suno·Udio)**: 유료 플랜에서 상업 라이선스 명시적. 무료 플랜은 비상업만

### 알고리즘·정책
- 메타·X·YouTube 모두 **AI 생성 콘텐츠 라벨링 정책** 적용 (2024~2026)
- 라벨 누락 시 노출 페널티 가능성
- **공식 가이드라인 준수 권장** — 약간의 후보정·내레이션이 들어가면 페널티 약함

### 자막 가독성
- AI 자동 자막은 정확하지만 **위치·크기·색이 영상과 충돌**할 수 있음
- 해결: 자막에 검정 배경 박스 또는 굵은 외곽선 추가. 9:16 화면 하단 1/4 안에 배치하되 인스타 UI 가림 영역(맨 아래 200px)은 피함

### AI 음성·아바타의 "기분 나쁜 골짜기"
- 너무 부드러운 AI 음성·완벽한 아바타 표정은 오히려 거부감
- 해결: 살짝 더듬거나 강세를 추가 (ElevenLabs는 SSML 마크업으로 조절 가능)

## 마무리 — "AI는 편집실 안에 들어왔다, 메시지는 여전히 사람"

PPT 편(5번)·SNS 이미지 편(6번)에서 반복한 원칙이 동영상에도 똑같이 통한다.

- **잡일은 AI**: 컷 생성·자막·하이라이트·BGM·자동 변환
- **판단은 사람**: 후크·메시지·길이·플랫폼 적합도·정서

특히 숏폼은 **첫 1초**가 모든 것을 결정하는데, 이 첫 1초의 후크는 도구가 골라주지 않는다. 사람이 "이 영상이 누구에게 무엇을 약속하는가"를 한 줄로 정의해야 한다.

도구를 익히는 시간은 줄여라 (한 도구 골라 1주일이면 충분). 그렇게 아낀 시간을 **무엇을 보여줄지 결정하는 데** 다시 써라. 결국 영상 만드는 일은 도구 싸움이 아니라 메시지 싸움이다.
