## 공통 스타일 가이드 (그리드 일관성용)

전체 10장에 공통으로 적용. **이 부분이 일관성의 핵심** — 어떤 도구를 쓰든 이 톤을 유지하세요.

### 컬러 팔레트
| 역할 | HEX | 사용처 |
|------|-----|--------|
| 메인 (잔디/숲) | `#1F4D2C` | 배경 메인 톤 |
| 보조 (진한 네이비) | `#0F2A3F` | 텍스트 위계 / 강조 박스 |
| 강조 (노랑) | `#F4C430` | 핵심 키워드 강조 / 번호 |
| 베이스 (오프화이트) | `#F4F1E8` | 본문 카드 배경 |
| 본문 텍스트 | `#1A1A1A` | 가독성 |

### 톤 매너 키워드 (모든 이미지 프롬프트에 공통 삽입)
- `editorial sports infographic style`
- `minimalist, clean layout`
- `morning light on golf course`
- `golf green and warm yellow accent palette`
- `not photorealistic faces, focus on silhouettes or anatomical diagrams`

### 폰트 가이드 (Canva·Photoshop 합성 시)
- **한글 제목**: Pretendard Bold 또는 나눔스퀘어 ExtraBold
- **한글 본문**: Pretendard Regular / Medium
- **숫자·영문**: Inter Bold / Söhne (없으면 Helvetica)
- **제목 크기**: 80~110pt
- **본문 크기**: 28~40pt
- **자간**: -25 ~ -50 (한글 제목은 살짝 좁게)

### 레이아웃 원칙
- **세이프 존**: 모든 가장자리에서 80px 안쪽 (인스타 UI 가림 방지)
- **상단·하단 여백** 충분히 (인스타 캡션·반응 버튼이 가리므로 핵심은 중앙)
- **한 카드 한 메시지** — 정보 욕심 금물
- **시리즈 표시**: 우측 하단에 `01 / 10` 같은 페이지 인디케이터

## 카드 1 — 표지: "장타의 비밀, X-Factor"

### 카피 (정확히 그대로 합성)
```
대제목:    장타의 비밀
부제목:    X-Factor

훅 (작게): 어깨와 골반의 회전 차이가
          클럽헤드 스피드를 만든다

하단:      골프 학습 시리즈 · 01
          @your_handle
```

### 이미지 프롬프트 (Midjourney / DALL-E 3 / FLUX 공통)
```
A dramatic silhouette of a golfer at the top of backswing,
viewed from behind, soft morning mist on a green fairway,
golden hour light from the right side,
editorial sports magazine cover style,
deep green and navy background with warm yellow accent light,
minimalist composition with strong negative space on the upper third,
square 1:1 ratio
```

### 레이아웃
- 이미지 풀블리드 (1080×1080 가득)
- 상단 1/3: 대제목 (흰색, 큰 글씨)
- 중앙 부제목 "X-Factor" — 노란색, 가장 굵게
- 하단 1/4: 어두운 그라데이션 오버레이 + 훅 텍스트 (오프화이트)
- 우측 하단: 시리즈 인디케이터 `01 / 10`

### 추천 도구
- **이미지**: Midjourney v7 (실루엣·아침 안개 톤이 가장 자연스러움)
- **합성**: Photoshop 또는 Canva

## 카드 2 — 정의: "X-Factor가 뭔가요?"

### 카피
```
제목:    X-Factor란?

본문:    백스윙 탑에서
        어깨 회전 각도 − 골반 회전 각도

예시:    어깨 90° 회전
        + 골반 45° 회전
        = X-Factor 45°

캡션:    'X'는 어깨와 골반 라인이
        정면에서 X자로 교차하는 데서 유래
```

### 이미지 프롬프트
```
Clean anatomical infographic illustration of a golfer
from behind at top of backswing,
showing shoulder rotation line and hip rotation line as
two distinct intersecting lines forming an X shape,
flat 2D illustration, technical diagram style,
deep navy background with yellow accent lines for the X,
white body silhouette with semi-transparent grid lines,
minimal, educational, no text in image,
square 1:1
```

### 레이아웃
- **상단 1/2**: 일러스트 (어깨·골반 라인 시각화)
- **하단 1/2**: 오프화이트 박스에 정의 텍스트
- "X-Factor 45°" 큰 노란 글씨로 강조

### 추천 도구
- **일러스트**: FLUX 또는 Recraft (인포그래픽 스타일에 강함)
- **합성**: Canva — Magic Resize 활용 (다른 카드와 톤 통일)

## 카드 4 — X-Factor Stretch: "진짜 파워의 비밀"

### 카피
```
키워드:    X-Factor Stretch

핵심:    중요한 건 정적 X-Factor가 아니라
        다운스윙 초기의 동적 변화다

설명:    백스윙 탑에서 45°였더라도
        다운스윙 시작에 골반이 먼저 풀리며
        일시적으로 50~55°까지 더 벌어진다

→ 장타자일수록 이 stretch 구간이 크다
```

### 이미지 프롬프트
```
Two-frame side-by-side illustration of a golfer's torso
from above, top-down view,
left frame: top of backswing with X angle measurement,
right frame: early downswing with hips rotating first
while shoulders stay back, showing larger angle,
labeled with measurement arcs in warm yellow,
deep navy background, clean technical diagram style,
square 1:1
```

### 레이아웃
- **상단**: 두 컷 비교 일러스트 (정적 vs Stretch)
- **하단**: 핵심 텍스트
- 화살표·각도 표시는 합성 단계에서 Canva로 추가 권장

### 추천 도구
- **일러스트**: DALL-E 3 (개념 다이어그램 표현이 안정적)
- **각도·화살표 오버레이**: Canva 또는 Figma

## 카드 5 — 적정 범위: "당신의 X-Factor는?"

### 카피 (이건 표 형태로 합성)
```
제목:    적정 범위 (우타 기준)

표:
┌────────────────┬───────────┐
│ 어깨 회전        │ 80°~100°  │
│ 골반 회전        │ 40°~55°   │
│ X-Factor (정적)  │ 35°~50°   │
│ Stretch (동적)   │ +5°~10°   │
└────────────────┴───────────┘

캡션:    개인의 흉추·고관절 가동성에 따라
        편차가 크다. 자기 가동범위 내에서
        최대화가 원칙.
```

### 이미지 프롬프트
```
Minimalist background image of an aerial top-down view
of a golf course green with subtle radial measurement
lines suggesting angle measurements,
deep green and cream tones, soft morning light,
no people, very subtle and abstract,
suitable as background for overlaid table text,
square 1:1
```

### 레이아웃
- **배경**: 흐릿한 골프코스 톱뷰 (텍스트 가독성 우선)
- **중앙 풀픽**: 오프화이트 카드에 표
- 표는 Canva에서 직접 작성 (이미지로 만들지 말 것 — 가독성 우선)

### 추천 도구
- **배경**: Midjourney 또는 FLUX
- **표 합성**: Canva (표 템플릿 사용)

## 그리드 미리보기 (인스타 프로필 9분할 셀)

10장이 캐러셀로 묶이면 프로필 그리드에는 첫 카드(표지)만 노출. 그래도 **시각 톤 일관성**을 위해 모든 카드가 같은 팔레트·여백·폰트 위계를 따라야 함.

```
인스타 프로필 그리드 (9칸 중 1칸 차지)
┌──────┬──────┬──────┐
│ [표지] │  ?   │  ?   │   ← 표지가 첫 칸. 그 옆은 다음 글
├──────┼──────┼──────┤    들로 채워지므로 시리즈 톤 통일이 매우 중요
│  ?   │  ?   │  ?   │
└──────┴──────┴──────┘
```
