볼은 **페이스 방향으로 출발한다**. D-Plane 글에서 다뤘듯 페이스가 출발 방향의 80~85%, 휨의 근본 원인의 절반 이상을 결정한다. 같은 시퀀스·같은 GRF·같은 X-Factor라도 페이스가 열려 있으면 슬라이스, 닫혀 있으면 훅이 난다. 페이스 컨트롤은 사실상 골프의 거의 모든 것이다. 이 글은 (1) 어떤 백스윙이 페이스를 열리게 만드는가, (2) 샬로잉과 페이스 열림의 관계, (3) 페이스 닫는 방법을 정리한다.

---

## 1. 어떤 백스윙이 페이스를 열리게 만드는가

페이스가 임팩트에서 열려 있는 골퍼의 거의 모두가 **백스윙 어딘가에서** 이미 페이스를 열어둔다. 임팩트 시점만 고치려는 시도가 잘 안 되는 이유. 백스윙의 5개 체크포인트를 보자.

### A. 그립

| 그립 형태 | 페이스 경향 |
|----------|------------|
| **Weak (약한 그립)** | V자가 턱·왼쪽 어깨를 가리킴 → 백스윙 중 페이스 자연스럽게 오픈 |
| **Neutral** | V자가 우측 귀 부근 → 페이스 중립 |
| **Strong (강한 그립)** | V자가 우측 어깨 → 페이스 자연스럽게 클로즈 |

대부분의 슬라이서는 **위크 그립**으로 시작한다. 손바닥이 위를 향하면 백스윙 탑에서 페이스도 같이 위(=오픈)를 향한다.

### B. Takeaway (시작 1m)

- **클럽 toe pointing straight up (9시 위치)** = 스퀘어 페이스
- **toe back/face up** = 페이스 오픈 (열림)
- **toe forward/face down** = 페이스 클로즈 (닫힘)

손목을 너무 일찍 회전시켜(예: 셋업에서 곧장 fan open) takeaway 1m 안에 페이스가 활짝 열리는 골퍼가 많다. 이 경우 백스윙 내내 페이스를 다시 닫을 기회는 거의 없다.

### C. 백스윙 탑 — 리드 손목

가장 결정적 체크포인트.

| 리드 손목 (왼손목, 우타) | 페이스 |
|------------------------|-------|
| **Cupped (extension)** — 손등이 위로 젖혀짐 | **오픈** |
| **Flat (neutral)** — 일직선 | **스퀘어** |
| **Bowed (flexion)** — 손등이 타깃 쪽으로 굽음 | **클로즈** |

투어에서 일관된 페이스 컨트롤로 유명한 선수(Dustin Johnson, Jon Rahm, Brooks Koepka)는 모두 **bowed wrist**를 쓴다. 입문자 대다수는 cupped → 페이스 오픈 → 슬라이스.

### D. 어깨 회전 평면

- **너무 평평(flat shoulder turn)** → 클럽이 안쪽으로 빨려 들어가며 페이스가 위(오픈)를 향함
- **너무 가파른 회전(steep)** → 클럽이 머리 위로 가며 페이스가 닫힘
- 신체 가동성에 맞는 자연스러운 회전이 우선 — X-Factor·키네마틱 시퀀스 글 참조

### E. 클럽 플레인

- **백스윙 plane이 너무 under(안쪽)** → toe pointing back, 페이스 크게 오픈 → 다운스윙에서 over-the-top 보상 강요
- **너무 across(위)** → 페이스 닫힘 + 다운스윙 슬라이드

→ 정리: 슬라이스의 진짜 원인은 **임팩트가 아니라 백스윙**에 거의 있다.

## 2. 샬로잉(Shallowing)과 페이스 열림의 관계

### 샬로잉이란

**샬로잉(Shallowing)** = 다운스윙 시작 직후 클럽이 백스윙 평면보다 **더 안쪽으로(평평하게) 떨어지는** 동작.

- 좋은 골퍼의 공통 특징 (TrackMan에서 다수 확인됨)
- 클럽 패스가 in-to-out으로 진입하게 만드는 핵심 메커니즘
- 입문자에게 권장되는 트랜지션 동작

### 샬로잉의 함정 — 페이스가 같이 닫히지 않으면 푸시 슬라이스

샬로잉만 단독으로 시도하면 오히려 구질이 나빠진다. 이유:

```
샬로잉만 = 클럽 패스가 in-to-out으로 바뀜
           BUT 페이스가 같이 닫히지 않으면 →
           Face open + Path right = Push Slice 또는 Big Push
```

D-Plane 글의 "Face-to-Path 차이가 휨을 결정"을 그대로 적용해 보면:
- Path 우측 (in-to-out)
- Face가 Path보다 더 우측으로 오픈 → **푸시 슬라이스**
- Face가 Path보다 좌측 → **드로우/훅** (이게 진짜 목표)

→ **샬로잉은 페이스 닫음과 반드시 한 쌍으로**.

### 샬로잉과 페이스 닫음을 동시에 만드는 3가지 메커니즘

#### ① Lead Wrist Flexion (DJ 메커니즘)
- 백스윙 탑에서 또는 트랜지션 시 리드 손목 굴곡
- 손목이 굽으면서 페이스가 클로즈됨
- 동시에 클럽이 자연스럽게 샬로잉됨
- 가장 정확하지만 손목 컨트롤 어려움

#### ② Trail Forearm Supination (전완 회전)
- 트레일 팔(우타: 오른팔) 전완이 시계 반대 방향(supination)으로 회전
- "피자 박스 받쳐 들기" 자세
- 클럽이 inside로 떨어지며 페이스가 동시에 클로즈
- 직관적이고 느낌 좋음

#### ③ Body Sequence 우선
- 키네마틱 시퀀스(골반 선행)만 정확하면 자연스럽게 샬로잉 + 페이스 클로즈 발생
- 손목·전완 의식 적게 가능
- 가장 안정적이지만 시퀀스가 망가지면 동시에 모든 게 무너짐

투어 프로의 페이스 컨트롤은 이 3가지가 **결합된 채로** 작동한다. 단일 동작에 의존하면 일관성 부족.

## 3. 페이스 닫는 방법

### A. 그립 손보기 (가장 빠른 효과)

- **Weak → Neutral 또는 Strong**으로 이동
- V자가 우측 어깨를 가리키도록
- 리드 손바닥이 페이스와 평행하지 않도록 — 페이스보다 약간 닫힌 방향으로 잡기
- 그립만 바꿔도 슬라이스의 50%가 잡히는 경우 흔함

### B. 백스윙 탑에서 Bowed Wrist 만들기

- 리드 손목을 굽혀(flexion) 손등이 타깃 쪽으로
- 백스윙 탑에서 손등이 평평하거나 약간 굽은 상태가 목표
- 갑자기 강하게 굽히면 부상 위험 — 점진적으로 도입

**드릴**:
- **Bowed Wrist 드릴**: 클럽 짧게 잡고 백스윙 탑 자세에서 손목 굽힌 상태로 멈춰 인지
- **거울 체크**: 백스윙 탑에서 거울 보며 손목 형태 점검

### C. Takeaway 페이스 매니지먼트

- 시작 1m에서 클럽 toe가 똑바로 위(스퀘어) 또는 약간 앞으로 기울어진 상태(closed)
- 손목을 일찍 회전시키지 말 것 — "손목보다 어깨가 먼저"

**드릴**:
- **9시 정지 드릴**: takeaway에서 클럽이 9시 위치(지면과 평행)에 왔을 때 멈춰 toe 위치 확인
- **One-piece Takeaway**: 어깨·팔·클럽이 한 덩어리로 움직이는 감각

### D. Trail Forearm Supination

- 다운스윙 시작 시 트레일 전완을 시계 반대 방향으로 회전
- 이전 글의 키네마틱 시퀀스 + 임팩트 다이내믹스와 결합

**드릴**:
- **피자 박스 드릴**: 트레일 손바닥에 피자 박스를 올린다고 상상하며 다운스윙 — 손바닥이 위를 향한 채 유지
- **클럽 가로 잡고 회전**: 클럽을 가로로 잡고 다운스윙 동작만 — 전완 회전 인지

### E. Body Rotation 시퀀스 (근본 해결)

- 골반 선행 → 흉곽 → 팔 순서로 회전
- 시퀀스만 정확하면 손목·전완 조작 없이도 페이스가 자연스럽게 클로즈
- 가장 견고한 페이스 컨트롤은 **시퀀스 기반**

**드릴**:
- Step Drill, Pump Drill (키네마틱 시퀀스 글 참조)
- **임팩트 후 회전 드릴**: 임팩트 직후 몸통이 멈추지 않고 계속 회전 → 페이스가 자연스럽게 클로즈

### F. 임팩트 백 (Impact Bag)

- 임팩트 가방을 치며 Forward Shaft Lean + Closed Face 동시 인지
- 가방을 친 자세로 정지 → 손목·페이스·골반 위치 확인
- 가장 직접적 임팩트 피드백

## 4. 페이스 vs 패스 — 균형이 핵심

페이스를 닫기만 하면 끝이 아니다. 페이스가 패스보다 **너무 많이 닫히면** 풀-훅·풀로 또 다른 문제.

| Face-to-Path | 구질 |
|-------------|------|
| Face = Path | Straight (직선) |
| Face가 Path보다 **2~4° 닫힘** | **드로우** ← 이상적 목표 |
| Face가 Path보다 **5° 이상 닫힘** | 훅·풀-훅 |
| Face가 Path보다 **2~4° 열림** | **페이드** ← 이상적 목표 (페이드 플레이어) |
| Face가 Path보다 **5° 이상 열림** | 슬라이스 |

→ 페이스 컨트롤의 목표는 **절대 스퀘어**가 아니라 **패스 대비 ±2~4°**.

## 5. 진단: 내 슬라이스/훅의 진짜 원인은?

```
슬라이스 (오른쪽으로 휨)
├─ 페이스 오픈 (vs Path) ← 거의 항상
│   ├─ 그립이 weak? → Strong/Neutral로
│   ├─ 백스윙 탑 cupped? → Bowed로
│   ├─ Takeaway에서 일찍 fan open? → 9시 정지 드릴
│   └─ 시퀀스 무너져 손이 던져짐? → 시퀀스 드릴
└─ Path 자체가 아웃-인? → 셋업·시퀀스 점검

훅 (왼쪽으로 휨)
├─ 페이스가 Path보다 너무 닫힘
│   ├─ 그립이 너무 strong? → 약간 약화
│   ├─ Takeaway에서 toe 너무 닫힘? → 9시 toe 위치 점검
│   └─ 다운스윙에서 손목 회전(roll) 과다? → 회전 자제
└─ Path가 너무 인-아웃? → 정상 패스로 회귀
```

## 6. 흔한 오해

| 오해 | 진실 |
|------|------|
| "샬로잉만 잘 하면 슬라이스 사라진다" | 페이스 닫음 동반 없으면 푸시 슬라이스로 악화 |
| "임팩트 직전에 손목 빠르게 회전(roll)해서 닫으면 된다" | 타이밍 컨트롤 어려워 일관성 부족. 백스윙 탑에서 이미 닫혀 있어야 안정 |
| "그립만 strong으로 바꾸면 끝" | 그립이 시작이지만 takeaway·백스윙 탑·시퀀스가 다 일관돼야 |
| "페이스는 항상 스퀘어가 정답" | 패스 대비 상대값이 본질. 절대 스퀘어는 환상 |
| "Bowed Wrist는 프로의 동작, 입문자는 따라하면 안 됨" | 점진적으로 적용하면 입문자에게도 가장 안정적 |

## 7. 정리

- 페이스 컨트롤은 골프의 거의 모든 것 — 출발 방향의 80~85% 결정
- 임팩트의 페이스 오픈은 거의 모두 **백스윙에서** 이미 결정됨
  - 그립 → Takeaway → 백스윙 탑(특히 리드 손목) → 어깨 회전 → 클럽 플레인
- **샬로잉은 페이스 닫음과 한 쌍으로** — 단독 시도 시 푸시 슬라이스 악화
- 페이스 닫는 방법
  - 그립 강화 (가장 빠름)
  - 백스윙 탑 Bowed Wrist
  - Takeaway 페이스 매니지먼트
  - Trail Forearm Supination
  - Body Sequence 기반 자동 클로즈 (가장 안정)
  - Impact Bag 직접 피드백
- 목표는 **절대 스퀘어가 아닌 패스 대비 −2~−4° (드로우)** 또는 **+2~+4° (페이드)**
